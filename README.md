# Electric-vehicals-analysis
# ⚡ EV DATA — Electric Vehicle Analytics Dashboard

## 📊 Overview
An interactive **Power BI dashboard** built to analyze electric vehicle (EV) data across models, manufacturers, and performance metrics.  
It integrates **SQL**, **Power Query**, and **data modeling** to deliver actionable insights on EV trends and efficiency.

---

## 🧠 Key Insights
- **Total Model Year Records:** 2M+  
- **Top Manufacturer:** AUDI  
- **Leading Model:** BMW 330E  
- **Legislative Districts by Make:** Tesla leads with ~10K entries, followed by Nissan and Chevrolet.  
- **Electric Range Analysis:** Model 3 shows the highest cumulative range (~24K).  
- **Utility Count by Model:** Model 3 and Model Y dominate with 163 and 152 counts respectively.  
- **Average Electric Range:** 998 miles (gauge visualization).

---

## 🛠️ Tools & Techniques

| Tool | Purpose |
|------|----------|
| **SQL** | Data extraction, cleaning, and aggregation from EV datasets |
| **Power Query** | Data transformation and loading into Power BI |
| **Power BI** | Dashboard design, DAX calculations, and interactive visuals |
| **Data Modeling** | Relationships between Make, Model, Range, and Utility metrics |

## 🚀 Outcome
This dashboard provides a **data-driven narrative** of the EV industry — helping analysts and stakeholders identify top-performing models, range efficiency, and manufacturer dominance.
